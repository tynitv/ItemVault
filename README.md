# ItemVault (Virtual Storage & Backpacks) 📦

**ItemVault** is a virtual player vault and backpack storage system for **Paper 1.21.x** servers.

---

## Commands & Permissions

- /pv - Open your virtual player vault (Permission: itemvault.use, Aliases: /vault)
